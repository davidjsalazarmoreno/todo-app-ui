# Todo App UI

## Prerequisites

- Node.js (v22.4.1 or higher)
- npm or yarn

## Installation

```bash
nvm install && nvm use
npm install
```

## Development

To start the development server:

```bash
npm run dev
```

## Building for Production

To create a production build:

```bash
npm run build
```

## Testing E2E

To run tests:

```bash
npm run cy:open
```

Note: you need the API and the UI running
